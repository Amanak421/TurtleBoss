TurtleBoss
============


.. image:: _static/Turtle_Boss.png
   :width: 300px
   :alt: Project Logo
   :align: center


This is the documentation for our team project for B3B33LAR. The goal of the assignment is to find a ball and two goal posts and score a goal between them. Assignment for the team project can be found `here <https://cw.fel.cvut.cz/wiki/courses/b3b33lar/tasks/scoregoal_cs>`_.

Usage
-----

The program can be started by running following command in the terminal.


.. code-block:: python


  python3 kick_goal.py

If any argument are added, the program will run in DEBUG mode.


.. code-block:: python


  python3 kick_goal.py DEBUG
