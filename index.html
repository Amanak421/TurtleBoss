TurtleBoss

This is the documentation for our team project for B3B33LAR. The goal of the assignment is to find a ball and two goal posts and score a goal between them. Assignment for the team project can be found here.

Usage

The program can be started by running following command in the terminal.

python3 kick_goal.py

If any argument are added, the program will run in DEBUG mode.

python3 kick_goal.py DEBUG
